Чтобы не забыть какие урлы у нас есть:

Что есть

default.aspx - все посты
default.aspx?cat=1&page=1 - просмотр какой то категории(cat=0 все категории)(page=1 пагер )
addpost.aspx - add post

favorites.aspx?page=1 - избранные(page=1 - пагер )
favorites.aspx?a=del&post=999 - удаляем из избранного текущего пользователя пост с айди 999
favorites.aspx?a=add&post=999 - добавляем в избранное текущего пользователя пост с айди 999

news.aspx?id=5 новость

mailsend.aspx?receiver=nick отправка сообщения
mailview.aspx?a=output просмотр списка отправленных сообщений
mailview.aspx          просмотр списка полученных сообщений
mail.aspx?id=1 		   просмотр одного сообщения

rss.aspx
